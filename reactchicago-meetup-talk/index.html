Use D3 to draw a path


						<svg id="d3-path"></svg>
						

						var path = d3.path();
						path.moveTo(50, 0);
						path.lineTo(100, 50);
						path.lineTo(0, 50);
						path.lineTo(50, 100);

						d3.selectAll("#d3-path")
						  .attr("width", 100)
						  .attr("height", 100)
						  .append("path")
						    .attr("d", path.toString())
						    .attr("stroke", "white")
						    .attr("fill", "none")

D3 with React


						render() {
						  var path = d3.path();
						  path.moveTo(50, 0);
						  path.lineTo(100, 50);
						  path.lineTo(0, 50);
						  path.lineTo(50, 100);

						  return (
						    <svg width="100" height="100">
						      <path d={path.toString()} stroke="white" "fill"="none"/>
						    </svg>
						  )
						}

Canvas has an imperative api

you get the 2d context and call different methods on the context to draw different shapes

context.rect to draw a rectangle or square

context.ellipse to draw an ellipse or a circle

context.arc and the works

A chart with multiple shapes

x and y axes have ticks as line and tick labels as text

You get the idea, any complex data viz, is composed of many simple svg elements
